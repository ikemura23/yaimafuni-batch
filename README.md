# Yaimafuni-Backend

## 環境設定

nodenvを使用

https://github.com/nodenv/nodenv

## 起動方法

```
npm run start
```

もしくは

```
node src/index.js
```

## node.jsのバージョン固定
`.node-version`ファイルにnode.jsのバージョン指定をしている。

## herokuのnodeバージョンを指定

`package.json`の`enginesにnode.jsのバージョンを指定できる。
https://devcenter.heroku.com/ja/articles/nodejs-support#specifying-a-node-js-version
